Árukereső - Megbízható bolt program integrálása Woocommerce rendszerhez

== Description ==

Ezzel a bővítménnyel össze lehet kapcsolni a Woocommerce áruházunkat az Árukereső - Megbízható bolt rendszerével.

Mindössze az Árukereső WebAPI kulcsot kell beilleszteni a bővítmény beállítás oldalán, melyet itt lehet beszerezni:
http://www.arukereso.hu/admin/TrustedSettings

Amennyiben a regisztráció során megadott email címmel rendelünk, az megjelenik a fentebbi oldalon a Technikai Segítség fül alatt található táblázatban.

== Installation ==

Telepítés folyamata:

1. Töltsd fel a bővítményt a Bővítmények > Új bővítmény > Új bővítmény feltöltése oldalon
2. Ezek után aktiváld
3. A beállítások oldalon add hozzá a WebApi kulcsod
4. Kész
